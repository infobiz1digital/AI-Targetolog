# Actions Log — kevin

Хронологічний лог усіх дій по проєкту: зміни бюджетів, паузи/запуски, нові кампанії/adset/ads, тести, висновки.

Формат запису:

```
- [YYYY-MM-DD HH:mm] @username: <дія> — <причина / результат>
```

Дописується автоматично командою `/action <project> <text>` боту в Telegram, а також worker'ом при детектуванні значимих змін у кабінеті.
